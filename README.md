📚 Library Management System (C)

A console-based Library Management System developed in C to manage basic library operations such as book records, members, and book issuing. The system uses a menu-driven interface and demonstrates core C programming concepts.

✨ Features

🔐 User Login Authentication

📖 Add New Books

📋 Display All Books

✏️ Update Book Details

❌ Delete Books

🔎 Search Books

👤 Add Members

📦 Issue Books

↩️ Return Books

⚠️ View Not Returned Books

⏰ View Overdue Books

📚 Check Available Book Copies

🛠 Technologies Used

C Programming Language

Standard C Libraries:

stdio.h

string.h

stdlib.h

📚 Concepts Used

This project demonstrates the use of:

Structures (struct)

Arrays

Functions

Loops

Conditional statements

String handling (fgets, strcmp, strcpy)

Basic data management in C

🖥 System Workflow

User logs into the system using a username and password.

The system displays a menu of library operations.

The user selects options to manage books, members, and issued records.

The system updates and displays the data accordingly.
